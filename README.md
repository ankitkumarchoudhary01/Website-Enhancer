# Motion Layer

A cinematic browser interaction engine that enhances normal websites with:

- Smooth motion physics
- Dynamic lighting
- Cursor glow
- Particle trails
- Glassmorphism overlays
- Velocity-based animations
- Smooth scrolling
- Theme-aware rendering

## Features

✨ Cursor Glow  
✨ Ripple Effects  
✨ Particle Trails  
✨ Dynamic Lighting  
✨ Glass Overlay  
✨ Smooth Scroll Physics  
✨ Velocity Motion Engine  
✨ Theme Detection  

## Tech Stack

- JavaScript
- CSS3
- Chrome Extension APIs
- GPU Accelerated Rendering

## Installation

1. Clone the repository
2. Open Chrome Extensions
3. Enable Developer Mode
4. Click "Load Unpacked"
5. Select the project folder
